## Project Description
This project is a custom **Grafana Panel Plugin** developed as part of the Final Project requirements.

The plugin builds successfully, loads inside Grafana, and renders a custom visualization panel.
As required, the plugin clearly displays the developer's name inside the panel UI.

**Developed by: Sudenur Tilla**

---

## Mandatory Requirements (Completed)
- Custom Grafana panel plugin
- Plugin builds successfully
- Plugin loads inside Grafana
- Plugin renders inside a dashboard
- Student name is visible inside the plugin UI

---

## Bonus Features Implemented
- Renders real data from Grafana queries
- Displays number of data series dynamically
- Configurable panel options:
  - Custom text input
  - Show / hide series count
- Improved UI layout and styling
- Optional dashboard JSON included
